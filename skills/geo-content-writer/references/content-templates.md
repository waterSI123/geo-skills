# Content Templates

All drafts should be Markdown with YAML-like frontmatter.

## Shared Frontmatter

```markdown
---
content_id: "CONTENT001"
asset_type: "comparison_article"
target_topic: "AI ad creative generator for ecommerce"
target_personas:
  - "ecommerce marketer"
target_market: "US"
target_language: "English"
source_gap_signals:
  - "competitor_recommended_client_absent"
source_risk_signals: []
recommended_url_slug: "/best-ai-ad-creative-generators-ecommerce"
needs_client_evidence: true
status: "draft"
---
```
