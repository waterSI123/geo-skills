# GEO Writing Rules

Write content for AI retrievability, entity clarity, and recommendation readiness.

## Core Goals

- Make the brand entity explicit.
- Make the product category explicit.
- Make target personas and use cases explicit.
- Make the official website/domain explicit.
- Put the brand into relevant comparison and category contexts.
- Include FAQ answers that are short, direct, and reusable by AI systems.
- Write for realistic buyer topic clusters, not for a single exact monitoring prompt.

## Writing Style

- Use clear headings.
- Use concise paragraphs.
- Prefer concrete category and use-case language.
- Avoid exaggerated claims.
- Avoid attacks on competitors.
- Use "may", "can help", and "is designed for" when evidence is limited.
- Avoid prompt-shaped copy that reads like it was created only to move a dashboard metric.

## Required Mentions

Every draft should mention:

- client brand
- official website or domain
- target topic
- target persona or buyer role
- target market or language when available

## Competitor Context

When competitors are included:

- Present them as alternatives or comparison context.
- Do not invent product features or performance claims.
- Use brief neutral language, such as "tracked competitor" or "alternative buyers may consider".

## FAQ Guidance

FAQ answers should:

- answer directly in the first sentence
- include the brand name
- include category language
- avoid unverifiable proof claims
